Explain the difference between inline, embedded, and external CSS.

Inline CSS is added directly to an HTML element. Embedded CSS is written inside a style tag within the HTML page. External CSS is stored in a separate file and linked to the HTML page.

Explain why you should only use external CSS.

External CSS should be used because it keeps HTML clean, makes it easy to change styles on many pages at once, and allows browsers to reuse and cache the CSS file.

What is a selector used for?

A selector is used to choose which HTML elements the CSS will style.
