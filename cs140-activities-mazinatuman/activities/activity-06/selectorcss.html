2. What is the diffrerence between an ID and a class?

An ID is unique for a single element, whereas a class can be applied to multiple elements.

3.What is the difference between a span tag and div tag

Span acts as an inline by default while a div behaves as a block element by default (it stacks and stretches all the way across the container)

4.What is the difference between a DIV tag and a Header

The div is a generic container used to group content or apply styles, but it has no specific meaning, whereas header, main, and footer are semantic elements that define the purpose of the content they contain. Header indicates introductory or navigational content, main identifies the primary content of the page, and footer marks footer content like legal or contact information. Using semantic elements improves accessibility, search engine optimization, and overall clarity of the page structure compared to using generic div containers.

5.What are the different ways to select a color?

Colors can be selected in different ways in web design. You can use a color name, a hexadecimal code, RGB values, RGBA values with transparency, HSL values, or HSLA values with transparency. Each method allows you to specify colors in a slightly different format, giving flexibility in how colors are defined and displayed.
